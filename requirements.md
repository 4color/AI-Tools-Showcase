# 系统需求文档

## 1. 前端应用
- 使用 Vue.js 和 Vite 构建。
- 提供用户界面以浏览各类 API。
- 功能包括：
  - API 的搜索功能。
  - 按 API 提供商过滤。
  - 分页浏览 API 列表。
  - 使用卡片布局显示 API 详情。
  - 响应式设计以适应不同屏幕尺寸。

## 2. 后端应用
- 使用 Java 和 Spring Boot 构建。
- 提供用于管理 API 和搜索 API 的 RESTful API。
- 功能包括：
  - 登录认证功能。
  - 请求和响应处理的数据传输对象 (DTO)。
  - 与 MySQL 数据库集成以存储 API 数据。

## 3. Docker 配置
- 使用 Docker Compose 管理服务。
- 服务包括：
  - 存储应用程序数据的 MySQL 数据库。
  - 用于 API 管理的后端服务。
  - 用户界面的前端服务。
- 提供了前端和后端应用程序的 Dockerfile。

## 4. 开发环境
- 前端使用 Node.js 和 npm 进行包管理和开发。
- 后端使用 OpenJDK 17 运行 Java 应用程序。

这些功能共同提供了一个完整的系统，用于管理和浏览一组 API，具有友好的用户界面和强大的后端支持。
